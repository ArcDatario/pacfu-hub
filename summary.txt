PACFU PORTAL FOR PSAU – SYSTEM SUMMARY (AI PROMPT READY)

SYSTEM OVERVIEW
PACFU Portal for PSAU is a web-based internal system designed exclusively for PACFU (faculty union) members and administrators of Pampanga State Agricultural University (PSAU). The system supports faculty management, communication, collaboration, document handling, surveys, elections, and notifications.

There are ONLY TWO USER ROLES:
1. Admin
2. Faculty

No other roles are allowed.

FACULTY FEATURES (TARGET BEHAVIOR)
--------------------------------
Faculty users can:
- Login securely
- Chat with co-faculty in assigned group chats
- Participate in polls and surveys created by Admin
- Vote in elections created by Admin
- Upload, view, and manage their own documents in a built-in drive-like page
- Receive in-app and push notifications

--------------------------------
ADMIN FEATURES (TARGET BEHAVIOR)
--------------------------------
Admin users can:
- Access an admin dashboard
- Create and manage faculty accounts
- Activate or deactivate faculty access
- Create and assign group chats
- Create announcements (with notifications)
- Create polls/surveys
- Create and manage elections
- View all uploaded faculty documents
- Manage financial records and receipts (future module)
- Generate system reports (future module)

SYSTEM GOAL
--------------------------------
To build a secure, role-based PACFU Portal for PSAU that centralizes:
- Faculty collaboration
- Communication
- Document management
- Surveys and elections
- Administrative control
